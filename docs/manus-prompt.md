# Manus用プロンプト — メリノちゃんナビゲーター画像6種

## 概要

VTuber情報サイト「メリノちゃんのVTuberガイド」(guide.merino-kobo.com) で使う
ナビゲーター画像を6種類生成してください。

記事内で「吹き出し」の横に表示する**バストアップ画像**です。

## リファレンス画像（添付）

| ファイル | 内容 |
|---------|------|
| `原画.png` | **マスター画像（最重要）**。服装・色・線のタッチはこれに完全準拠 |
| `merino_neutral.png` | 通常表情の参考 |
| `merino_presenting.png` | 手を広げたポーズの参考 |
| `merino_wink.png` | ウインクの参考 |
| `merino_surprised.png` | 驚き表情の参考 |
| `merino_thinking.png` | 考え中ポーズの参考 |
| `merino_waving.png` | 手を振るポーズの参考 |
| `merino_pointing_up.png` | 人差し指を立てるポーズの参考 |

## 生成する画像 6枚

### 1. merino-explain.png
- **ポーズ**: 手を広げて説明している（presenting に近い）
- **表情**: にこやかで親しみやすい笑顔、口を少し開けて話している感じ
- **用途**: デフォルトの解説シーン。一番多く使う画像

### 2. merino-surprise.png
- **ポーズ**: 片手を口元に当てるか、両手を頬の横に持ってくる
- **表情**: 目を見開いて驚いている。口は「お！」の形
- **用途**: 意外な事実、注意喚起のコールアウト

### 3. merino-recommend.png
- **ポーズ**: 片手で指差し（画面の右側＝読者から見て商品やリンクの方向を指す）
- **表情**: ウインク＋にっこり笑顔
- **用途**: おすすめ商品紹介、CTA（BOOTH/nizimaへの誘導）

### 4. merino-thinking.png
- **ポーズ**: 片手を顎に当てて考え中
- **表情**: 少し斜め上を見ている、穏やかだがやや真剣
- **用途**: 比較検討、メリット・デメリット解説

### 5. merino-happy.png
- **ポーズ**: 片手を振っている（手のひらをこちらに向けて）
- **表情**: 満面の笑み、目が少し細くなるくらいの笑顔
- **用途**: まとめ、成功、ポジティブな締めくくり

### 6. merino-point.png
- **ポーズ**: 人差し指を立てている（☝️のジェスチャー）
- **表情**: 少しキリッとした自信のある笑顔、「ここ大事！」という感じ
- **用途**: 重要ポイント、ワンポイントアドバイス

## 画像仕様

| 項目 | 指定 |
|------|------|
| **構図** | バストアップ（胸から上）。全身ではない |
| **サイズ** | 正方形 1024×1024px |
| **背景** | 鮮やかなマゼンタ単色 `#FF00FF`（後で透過処理するため） |
| **ファイル形式** | PNG |
| **向き** | 基本的にやや左向き（読者と対面する感じ）。recommend のみ右方向を指差し |

## キャラクターデザイン厳守事項

原画.png を参照し、以下を完全に再現すること：

- **髪**: 肩くらいのベージュ/クリーム色のウェーブボブ
- **目**: 大きな黄金色の瞳
- **頭の装飾**: 緑色の三角フード（先端が尖っている）
- **ケープ**: 緑色のケープ（肩にかかるパフスリーブ状、黄色い刺繍あり）
- **首元**: 小さな赤いリボン（蝶結び）
- **服**: 白いエプロン（胸から下に広がる）
- **アートスタイル・色使い・線の太さ**: 原画と完全に同一

### 禁止事項
- ❌ 牙・八重歯を描かない
- ❌ 襟を追加しない（原画に襟はない）
- ❌ ヘッドホン・イヤホンを付けない
- ❌ 原画にない装飾品を追加しない

## 生成後の後処理（こちらで実施）

1. マゼンタ背景を透過処理
2. 上下30pxクリップ（ウォーターマーク除去）
3. 512×512にリサイズ
4. `public/images/merino/` に配置
